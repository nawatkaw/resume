HelpLah

Personal Database driven project

An online database-driven web application that allows users to input their reviews about food stalls in University Town (UTown) in NUS.

A personal project working over the span of 1.5 months, using PostgreSQL, Python, HTML, CSS, and Django Framework.

My Motivation

I was motivated by seeing how there are many visitors in UTown. I feel that it would be great for these visitors to make an informed decision on what food should they eat in UTown by looking at the recommendation and reviews done by other users. Hence, anybody can critique!

Application’s features

This application has 2 main features – food stall search engine in UTown, and food stall review system.

The users are first direct to the main landing page as shown in the figure below. The users are asked to sign into their account, if not they are also allowed to register a new account too.

welcome --> log in
Log in --> Main page

Upon logging in, the users are re-direct into the main page which is the search engine for the database. This database are extracted from the database in PostgreSQL. The users are allowed to choose their search by name, by location or by types of food, and the result are displayed in the following page.

Main Page --> Search Result
Search Result --> Shop detail

By clicking at the stalls the users can see their search result showing the stall’s name, its location, type of food, the operating hours.

Shop detail --> Review

The users can choose their desired stall with all their reviews of that stall written. The users are also give the option to write a review for their respective stall.

Review successful
